# Screens

The pre-game flow runs linearly: Splash, then Main Menu, then three setup steps (scenario, crew, planet), then one final placement step before the session begins. In-game is a separate surface; the player reaches it either via "Continue" from the main menu or at the end of the setup flow.

## Pre-game flow (in order)

- [Splash](./splash.md) — boot sequence and loading screen; transitions to Main Menu when ready
- [Main Menu](./main-menu.md) — top-level navigation: new game, continue, load, settings, credits, exit
- [Scenario select](./scenario-select.md) — step 01 of 03; choose the expedition scenario (difficulty, party size, starting conditions)
- [Party selection](./party-selection.md) — step 02 of 03; read each survivor's dossier before committing
- [World generation](./world-generation.md) — step 03 of 03; configure and generate the planet
- [Landing site](./landing-site.md) — final approach; click the planet surface to set the descent vector

## In-game

- [In-game](./in-game.md) — the colony HUD: world view, colonist roster, minimap, storage, task queue, command bar, and colonist dossier dialog
